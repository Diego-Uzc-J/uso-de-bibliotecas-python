# Portafolio de Fundamentos de Ciencia de Datos en Python

¡Hola! Soy desarrollador Python actualmente enfocandome en Análisis de Datos (Nivel Junior). En este repositorio presento scripts prácticos diseñados para demostrar el manejo, manipulación y visualización de datos utilizando las bibliotecas esenciales del ecosistema.

## Contenido del Portafolio

*   **[1. NumPy (Álgebra Lineal y Matrices)](./01_numpy_demo/):** Demostración de operaciones matemáticas eficientes, creación de arreglos y manipulación de matrices.
*   **[2. Pandas (Análisis y Manipulación de Datos)](./02_pandas_demo/):** Limpieza de datos, filtrado, agrupación y manejo de DataFrames a partir de archivos reales.
*   **[3. Matplotlib (Visualización Gráfica)](./03_matplotlib_demo/):** Creación de gráficos estadísticos limpios, personalizados y profesionales para comunicar hallazgos.

## Requisitos Previos

Para ejecutar cualquiera de estos scripts, necesitas tener instalado Python 3 y las dependencias del proyecto. Puedes instalarlas todas juntas ejecutando:

```bash
pip install -r requirements.txt
```

## Comandos de ejecución:
```python3
python3 01_numpy_demo/practicas_numpy_usos_basicos.py 

python3 02_pandas_demo/practicas_pandas_usos_basicos.py

python3 03_matplotlib_demo/practicas_matplotlib_usos_basicos.py
```
